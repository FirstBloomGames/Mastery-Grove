HOW THE TREE GROWS

Every finished run feeds its own tree. Tree Total sets growth; each mode keeps its own Best. Standard Night Garden alone can prove the Lantern Willow Mastery Seed.

THE WEAVE

  1. Glide close to a flower and begin weaving.
  2. Touch at least two other flowers to pin the glowing thread.
  3. Return to the first golden flower to seal the loop.
